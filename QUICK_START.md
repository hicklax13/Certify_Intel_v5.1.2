# Certify Intel - Quick Start Guide

## One-Click Launch (Recommended)

**Double-click `Start_Certify_Intel.command`** - that's it!

The app will:
1. Install dependencies (first run only)
2. Start the server
3. Open your browser to http://localhost:8000

## System Requirements

- **macOS**: Apple Silicon (M1/M2/M3/M4) or Intel
- **Python 3.9+**: Download from https://www.python.org/downloads/ if needed

## First Time Setup

1. Download and unzip `Certify_Intel_v5.1.2.zip`
2. Double-click `Start_Certify_Intel.command`
3. If you see a security warning:
   - Right-click the file
   - Select "Open"
   - Click "Open" in the dialog
4. Wait for dependencies to install (first run takes ~1 minute)
5. Browser opens automatically
6. Login with credentials above

## Stopping the Server

- Close the Terminal window, OR
- Press `Ctrl+C` in the Terminal

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "Python not found" | Install Python from python.org |
| "Port 8000 in use" | The script auto-kills existing servers |
| Security warning | Right-click → Open → Open |
| Browser doesn't open | Manually go to http://localhost:8000 |
