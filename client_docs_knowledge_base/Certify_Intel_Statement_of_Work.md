# Statement of Work (SOW)

## Certify Intel — Competitive Intelligence Platform

**Document Version:** 1.0
**Effective Date:** January 16, 2026
**Prepared For:** Certify Health (Internal)

---

## 1. Project Overview

**Project Name:** Certify Intel
**Duration:** January 16 - February 9, 2026 (24 days)
**Objective:** Deliver an automated competitive intelligence system that monitors 40+ healthcare IT competitors and provides actionable insights via Excel dashboard.

---

## 2. Scope of Work

### 2.1 In Scope

| Deliverable | Description |
|-------------|-------------|
| **Excel Dashboard** | Production-ready workbook with competitor data, charts, and analysis views |
| **Cloud Backend** | Automated scraping and AI extraction system |
| **Data Collection** | 32 standardized data points per competitor |
| **Alert System** | Email notifications on critical competitor changes |
| **Documentation** | User guide and technical documentation |

### 2.2 Out of Scope

- Web application / custom UI (future phase)
- Real-time monitoring (weekly refresh only)
- Competitors outside healthcare IT
- Historical data prior to project start

## 7. Assumptions

1. Team members available as scheduled
2. Access to OpenAI API (GPT-4) provided
3. Cloud hosting (AWS/Azure) provisioned
4. Competitor websites accessible for scraping
5. Existing competitor research (CSVs) accurate and usable

---

## 8. What Certify Health Will Receive

Upon completion, Certify Health will receive:

1. **Excel Dashboard** — Production-ready workbook with:
   - Competitor overview (40+ companies)
   - Pricing comparison matrix
   - Feature comparison matrix
   - Market segment analysis
   - Change log / alert history

2. **Automated Backend** — Cloud-hosted system that:
   - Scrapes competitor websites weekly
   - Extracts data using GPT-4
   - Updates Excel via Power Query
   - Sends email alerts on changes

3. **Documentation** — Including:
   - User guide for Excel dashboard
   - Technical documentation for backend
   - Data dictionary (32 fields)
